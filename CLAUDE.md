# pv-bess-model

Single-file PV+BESS dynamic visual prototype. All code lives in `index.html`.

## GitHub
Remote: `git@github.com:adrianmlhood/pv-bess-model.git`
Branch: `master`

## Pushing changes
SSH doesn't work inside the Claude Code sandbox — always ask the user to run:
```
git push
```
from their terminal after committing. The remote is already configured.

## Project notes
- No build step — open `index.html` directly in a browser to test
- All JS, CSS, and HTML are inline in `index.html`
